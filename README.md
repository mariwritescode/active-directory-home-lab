# Active Directory Home Lab

## Overview

This project demonstrates the deployment and administration of a Windows Server Active Directory environment in VirtualBox.

Domain Name:
RockyPetSupply.local

## Technologies Used

- Windows Server
- Active Directory Domain Services (AD DS)
- DNS
- VirtualBox
- Windows Client
- Group Policy
- User and Group Management

## Project Objectives

- Deploy a Domain Controller
- Configure DNS
- Create Organizational Units (OUs)
- Create and manage users
- Create security groups
- Reset passwords
- Disable and lock accounts
- Join a workstation to the domain

## Environment

### Domain Controller
- Hostname: RPS-SERVER01
- Role: Domain Controller / DNS Server

### Workstation
- Hostname: RPS-WK01
- Joined to RockyPetSupply.local

## Skills Demonstrated

- Active Directory Administration
- DNS Configuration
- User Account Management
- Group Management
- Domain Join Operations
- Troubleshooting Authentication Issues
- Windows Server Administration
